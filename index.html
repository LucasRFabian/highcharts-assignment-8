Steam Hardware Survey Results

by Lucas Fabian

The Steam Hardware Survey is a voluntary survey given to all of the users of Steam, the most popular video game distribution platform for desktop computers. The survey collects several datapoints about the hardware that users are using to run Steam software, including CPU and GPU data, monitor resolution, and operating system.

Above is the CPU marketshare for PC users by manufacturer from 2012 to 2022. Intel is consistently leading AMD in the CPU market. The annual marketshare is taken from the survey publsihed on January first from each year, with the exception of 2022 which uses the datapoint from February first. The Steam Hardware Survey website only retains data from the current year, but using tools like the Wayback Machine, GitHub user myagues built a database that stores the historical results of the survey. Using this database, myagues parsed some of the data in charts available on ObservableHQ.

Above is the amount of free hard drive space that Steam users have. This datapoint is particularly pertinent for Steam as video game filesizes have been radically increasing in recent years. The presence or lack of free drive space, at least for Steam users, is usually indicative of the size of a user's downloaded game library.
